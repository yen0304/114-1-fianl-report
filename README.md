# 台灣縣市人口敘述統計分析

本專案使用 Python 和 Jupyter Notebook 對台灣各縣市民國 104-113 年的人口資料進行敘述統計分析。

## 專案結構

```

├── population_analysis.ipynb          # 主要分析筆記本
├── 程式用-縣市人口按性別及五齡組 - 工作表1.csv  # 原始資料
├── fonts/                            # 中文字型資料夾
│   └── Noto_Sans_TC/                # Noto Sans TC 字型家族
│       ├── static/                  # 靜態字型檔案
│       │   ├── NotoSansTC-Regular.ttf
│       │   ├── NotoSansTC-Bold.ttf
│       │   └── ... (其他字重)

│       └── README.txt               # 字型說明
├── pyproject.toml                     # uv 專案配置
├── uv.lock                           # 套件鎖定檔
└── README.md                         # 說明文件
```

## 分析內容

本筆記本包含以下分析：

### 1. 資料讀取與清理
- CSV 資料匯入
- 數值欄位清理（移除千分位符號）
- 計算 65 歲以上人口和老年人口比例

### 2. 基本敘述統計
- 總人口的統計量（平均數、中位數、標準差、偏度、峰度）
- 老年人口比例的統計量
- 各年齡層的統計摘要

### 3. 按縣市分組分析
- 各縣市平均人口排名
- 各縣市老年人口比例排名
- 各縣市統計摘要（平均、中位數、標準差等）

### 4. 按年份分組分析
- 歷年總人口趨勢
- 歷年人口成長率
- 歷年老年人口比例變化

### 5. 視覺化分析
包含多種圖表：
- 直方圖：總人口分布、老年人口比例分布
- 盒鬚圖：識別離群值
- 折線圖：歷年趨勢
- 長條圖：縣市排名、年齡層分布
- 熱力圖：各縣市歷年老年化趨勢
- 散布圖：變數相關性

### 6. 進階統計分析
- 變異係數（CV）分析
- 相關性矩陣和熱力圖
- 四分位數分析
- 離群值檢測

### 7. 結論摘要
- 整體概況統計
- 極值分析
- 趨勢分析

## 使用的套件

- **pandas**: 資料處理和分析
- **numpy**: 數值計算
- **matplotlib**: 基礎繪圖
- **seaborn**: 進階統計視覺化
- **jupyter**: 互動式筆記本環境

## 資料說明

資料來源：台灣各縣市人口統計（民國 104-113 年）

欄位包括：
- 區域別：22 個縣市
- 年份：104-113 年（共 10 年）
- 總人口
- 各年齡層人口：65～69、70～74、75～79、80～84、85～89、90～94、95～99、100以上
- 65歲以下人口
