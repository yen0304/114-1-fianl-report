# 台灣各縣市高齡人口與長照資源供需分析

## 一、研究背景與動機

### 研究背景
近年來，台灣快速邁入超高齡社會，高齡人口比例持續攀升。根據內政部人口統計，我國自民國109年起自然增加率轉為負值，顯示出生人口已無法支撐死亡人口，人口老化已成為不可逆的趨勢。

在高齡人口不斷成長的情況下，長期照顧需求急速擴張。政府推動長照2.0政策雖大幅擴增服務項目與供給量，但受限於地理分布、人口結構差異、偏鄉交通不便，以及各縣市老化速度不同，全國長照資源仍呈現供需不均與地區差異顯著的現象。

部分都市區人口密度高、服務使用率高，容易出現「需求大於供給」；偏鄉或山區則因面積大、機構少、交通距離長，造成「可近性不足」的長照服務缺口。為了有效配置有限的長照資源，必須透過統計分析，全面檢視各行政區的高齡人口結構、長照設施分布與可能的供需落差，以作為政策推動與資源配置的客觀依據。

---

## 二、研究目的

本研究旨在透過完整的量化資料與統計方法，系統性檢驗台灣各縣市長照資源配置是否達成供需平衡、改善高齡者生活品質與降低長照壓力的目的。研究目標包括：

### (一) 檢驗高齡化程度與長照資源配置的關係
1. 分析老年人口比例與長照據點數量的相關性
2. 探究人口密度對長照資源配置的影響
3. 評估不同地理區域（都市 vs. 偏鄉）的資源配置差異
4. 檢定高齡化程度是否能有效預測長照資源需求

### (二) 評估長照資源配置對服務充足度之改善效果
1. 計算各縣市實際資源充足度與理論需求量的差距
2. 判斷哪些縣市的資源配置成功滿足高齡人口需求
3. 探討每千位高齡人口據點數是否有效反映服務可近性
4. 比較 A、B、C 三級據點在不同縣市的配置效率

### (三) 探討地理因素對長照服務可近性的影響
1. 分析土地面積與長照據點密度的關係
2. 觀察偏鄉大面積縣市的服務可近性問題
3. 評估人口密度與服務密度的匹配程度
4. 識別「高齡化高但資源少」的資源不足熱點

### (四) 提供政策建議與實務應用方向
1. 改善長照資源配置的評估機制（例如：加入地理可近性指標）
2. 評估偏鄉交通補貼、據點增設等配套措施的重要性
3. 協助政府在人口老化趨勢下優先配置長照資源
4. 提出資源配置需搭配的配套方案

## 三、研究方法

本研究採用多種量化統計方法與資料處理手法，從描述性到推論性分析全面檢視長照資源配置之影響。

### (一) 資料蒐集與來源

| 資料項目 | 資料來源 | 時間範圍 | 用途 |
|---------|---------|---------|------|
| 縣市人口按性別及五齡組 | 內政部戶政司 | 104-113年 | 計算高齡人口比例與結構 |

### (二) 分析方向

本研究從四個面向進行分析：

#### 1. 高齡人口結構與長照資源分布之分析
- 各縣市老年人口比例的描述性統計
- 各縣市長照 A、B、C 據點數量統計
- 人口密度與土地面積的地理特徵分析

#### 2. 長照資源充足度與服務密度之分析
- 每千位高齡人口據點數的計算與比較
- 每平方公里據點數的空間密度分析
- 資源充足度的縣市排名與分組

#### 3. 長照資源供需落差之分析
- 計算各縣市理論所需據點數
- 評估實際據點數與所需據點數的缺口
- 識別資源不足與資源過剩的區域

#### 4. 高齡化程度與資源配置之關聯分析
- 老年人口比例與資源密度的相關分析
- 人口密度、土地面積對資源配置的影響
- 資源不足熱點的多維度交叉比對

### (三) 資料處理方法

#### 1. 資料清理與標準化處理
- **數值格式標準化**：移除千分位符號（,）、空白字元，統一轉換為浮點數
- **縣市名稱標準化**：去除空白、全形空白（　），統一縣市名稱格式
- **縣市代碼對照**：建立22縣市代碼對照表，處理不同資料源的編碼差異
- **欄位名稱清理**：使用 `str.strip()` 移除欄位名稱前後空白

#### 2. 缺失值處理策略
- **長照據點缺失值**：未有據點的縣市以 0 填充（`fillna(0)`）
- **資料合併策略**：使用 left join 保留所有縣市資料，確保分析完整性
- **異常值檢查**：檢查總人口、65歲以上人口等關鍵變數的合理性

#### 3. 衍生變數計算
- **高齡人口計算**：65歲以上 = Σ(65～69, 70～74, ..., 100以上)
- **老年人口比例**：(65歲以上人口 / 總人口) × 100
- **人口密度**：總人口 / 土地面積(km²)
- **每千位高齡人口據點數**：(總據點數 / 65歲以上人口) × 1000
- **每平方公里據點數**：總據點數 / 土地面積(km²)

#### 4. 供需指標定義化
- **合理服務標準**：採用全國平均「每千位高齡人口據點數」作為基準
- **所需據點數**：(65歲以上人口 × 全國平均服務率) / 1000
- **服務缺口**：所需據點數 - 實際總據點數
- **缺口比例**：(服務缺口 / 所需據點數) × 100

#### 5. 資料標準化與指標正規化
- **Z分數標準化**：`Z = (X - μ) / σ`，用於老化程度與資源充足度的標準化
- **優先設置分數**：老化程度Z分數 - 資源充足度Z分數
- **Min-Max正規化**：用於雷達圖多維度指標比較（標準化至0-1範圍）

#### 6. 資料整合與彙總
- **據點類型彙總**：按縣市名稱與據點類型（A/B/C）進行 groupby 統計
- **多資料源合併**：整合人口資料、長照據點資料、土地面積資料
- **年度篩選**：提取民國113年資料作為橫斷面分析

### (四) 統計分析方法

#### 1. 描述性統計分析（Descriptive Statistics）
- **目的**：掌握各縣市高齡人口結構與長照資源現況
- **統計量**：平均數、標準差、最大值、最小值、中位數
- **應用**：計算全國平均老年人口比例、平均據點數、平均人口密度
- **產出**：各縣市統計摘要表、排名表
- **輸出表格**：表七、各縣市主要指標統計表（含總人口、65歲以上、老年人口比例、總據點數、每千位高齡人口據點數、人口密度）

#### 2. 皮爾森相關係數分析（Pearson Correlation）
- **目的**：衡量高齡化程度與長照資源配置之線性相關性
- **分析變數對**：
  - 老年人口比例 vs. 每千位高齡人口據點數
  - 人口密度 vs. 每平方公里據點數
  - 65歲以上人口 vs. 總據點數
- **解讀標準**：
  - |r| > 0.7：強相關
  - 0.4 < |r| < 0.7：中度相關
  - |r| < 0.4：弱相關
- **檢定**：計算 p-value 檢定相關性顯著性（α = 0.05）
- **資料處理**：過濾 NaN 和無限值,僅對有效樣本進行分析
- **輸出表格**：表一、皮爾森相關係數分析統計表（含樣本數、r值、p值、顯著性標記）

#### 3. 散佈圖與趨勢線分析（Scatter Plot + Trendline）
- **目的**：視覺化檢視資料是否呈現線性或群聚型態
- **分析維度**：
  - X軸：老年人口比例
  - Y軸：每千位高齡人口據點數
  - 氣泡大小：高齡人口數
  - 顏色：服務缺口（色階編碼）
- **四象限分析**：
  - 右上：高老化+高資源
  - 右下：高老化+低資源（資源不足熱點）
  - 左上：低老化+高資源
  - 左下：低老化+低資源
- **迴歸線視覺化**：標註迴歸方程式與 R² 值於圖表中

#### 4. 簡單線性迴歸（Simple Linear Regression）
- **目的**：建立「高齡化程度 → 長照資源需求」預測模型
- **迴歸模型**：
  ```
  模型1: 每千位高齡人口據點數 = β₀ + β₁ × 老年人口比例 + ε
  模型2: 總據點數 = β₀ + β₁ × 65歲以上人口 + ε
  ```
- **檢定項目**：
  - 斜率顯著性檢定（t-test）
  - 截距顯著性檢定
- **模型評估**：殘差分析、常態性檢定
- **輸出表格**：
  - 表二、表五：線性迴歸統計表（含 Multiple R, R², 調整後 R², 標準誤, 觀察值個數）
  - 表三、表六：ANOVA表（含 df, SS, MS, F, 顯著性F）
  - 表四：線性迴歸係數表（含係數、標準誤、t統計、p-value）
- **視覺化圖表**：散佈圖搭配迴歸線,標註迴歸方程式與 R²

#### 5. R² 解釋力分析（Coefficient of Determination）
- **目的**：檢驗高齡化程度能解釋多少長照資源配置的變異
- **計算公式**：R² = 1 - (SSresidual / SStotal)
- **解讀標準**：
  - R² > 0.7：模型解釋力強
  - 0.4 < R² < 0.7：中等解釋力
  - R² < 0.4：解釋力弱
- **調整R²**：考慮變數數量的影響

#### 6. 供需落差評估（Supply-Demand Gap Analysis）
- **目的**：量化各縣市長照資源不足或過剩的程度
- **評估標準**：
  - 服務缺口 > 0：資源不足，需增設據點
  - 服務缺口 = 0：供需平衡
  - 服務缺口 < 0：資源充足或過剩
- **分級標準**：
  - 嚴重不足：缺口比例 > 30%
  - 輕微不足：10% < 缺口比例 ≤ 30%
  - 平衡：-10% ≤ 缺口比例 ≤ 10%
  - 過剩：缺口比例 < -10%

#### 7. 多維度交叉比對分析（Multi-dimensional Cross Analysis）
- **目的**：整合多項指標，識別資源配置的優先順序
- **分析方法**：
  - 標準化評分法（Z-score）
  - 熱點識別（雙條件篩選）
  - 優先設置分數排名
- **熱點定義**：
  - 條件1：老年人口比例 > 全國平均
  - 條件2：每千位高齡人口據點數 < 全國平均
- **產出**：優先設置建議排名、資源不足熱點清單
